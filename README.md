# TodoApp with React and Redux

TodoApp with React and Redux is a task management application built using React.js for the frontend user interface and Redux for state management.
![Screenshot 2024-04-02 222558](https://github.com/Komalyadav2021/Todoapp/assets/102368062/78ad39a7-e20f-415d-92b7-54057d136a6d)


## Features

- Add tasks with a title
- Edit existing tasks
- Delete tasks
- Filter tasks based on status (completed/uncompleted)
- Search tasks by title or description
- Responsive design for desktop use

## Technologies Used

- React.js and redux
- HTML5
- CSS3
- JavaScript
- Bootstrap
