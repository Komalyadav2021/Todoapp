# TodoApp with React and Redux

TodoApp is a modern task management application designed to help you stay organized and productive. Built with cutting-edge technologies including React.js, Redux, HTML5, CSS3, JavaScript, and Bootstrap, this app offers a seamless user experience with robust features.
![Screenshot 2024-04-02 222558](https://github.com/Komalyadav2021/Todoapp/assets/102368062/78ad39a7-e20f-415d-92b7-54057d136a6d)


## Features

Intuitive Task Management:
1. Add Tasks with Ease: Quickly add tasks by providing a title, ensuring effortless task creation.
2. Edit Existing Tasks: Easily modify task details, ensuring your task list stays up-to-date with your evolving needs.
3. Delete Tasks: Remove unwanted tasks effortlessly, maintaining a clutter-free task list.
4. Filter Tasks by Status: Stay organized by filtering tasks based on their status (completed/uncompleted), allowing you to focus on relevant tasks at any given time.

## Technologies Used

- React.js and redux
- HTML5
- CSS3
- JavaScript
- Bootstrap
